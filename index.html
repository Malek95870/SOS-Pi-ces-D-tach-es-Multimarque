MOD.ADMIN

Panneau Administrateur

§01

Stockage des fichiers LOCAL

URL racine où sont hébergés les fichiers Excel / PDF (NAS, serveur web, CDN, Google Drive public, etc.). Si vide, les fichiers sont cherchés en local à côté du HTML.
Exemple : https://mon-serveur.com/sav/ — le chemin de fichier sera ajouté à cette racine.

§02

Marques disponibles 00

Ajouter de nouvelles marques pour étendre l'outil au-delà d'ECOVACS (Roborock, Dreame, iRobot, Samsung, etc.). Chaque marque possède son propre catalogue de modèles. Les marques intégrées (ECOVACS) ne peuvent pas être supprimées.

§03

Ajouter un nouveau modèle

Les modèles ajoutés ici sont stockés localement dans le navigateur et apparaissent dans la marque/section choisie avec un repère ★.

§04

Modèles personnalisés 00

Modèles ajoutés via ce panneau. Les modèles intégrés ne peuvent pas être supprimés.

§05

Raccourcis fichiers 00

Ajoute des liens directs vers n'importe quel fichier Excel, PDF ou autre document (manuels, fiches techniques, procédures, etc.) — visible dans la section Documents & Raccourcis de la marque active.
Formats acceptés : https://... · file://... · C:\Dossier\fichier.pdf · \\serveur\partage\fichier.xlsx · chemin relatif.

§06

Générer un nouveau fichier HTML

Crée une copie autonome du fichier HTML avec la configuration actuelle intégrée par défaut : URL racine + marques personnalisées + modèles personnalisés + raccourcis. Utile pour distribuer une version pré-configurée à d'autres techniciens, faire une sauvegarde, ou figer la base de référence.

§08
